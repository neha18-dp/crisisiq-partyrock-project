# CrisisIQ - Emergency Preparedness & Response Simulator

AWS PartyRock Hackathon Project

## Overview

CrisisIQ is an AI-powered emergency preparedness simulator developed during the AWS PartyRock "Hack the Future with AWS" Hackathon.

The system generates crisis scenarios, severity assessments, response recommendations, and role-based guidance for civilians and government authorities.

## Features

- Crisis scenario generation
- Severity assessment
- Response recommendations
- Government authority guidance
- Civilian action plans
- AI-powered insights

## Technologies Used

- AWS PartyRock
- Foundation Models
- Generative AI
